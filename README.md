# Tech Blog Spot

 ## Description
  The Tech Blog Spot is an application built a place where developers can publish their blog posts and comment on other developersâ€™ posts as well.  This application follows the MVC paradigm in its architectural structure, uses Handlebars.js as the template language, Sequelize as the ORM, and the express-session npm package for authentication.
